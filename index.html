Come and contribute to Anubis

Rust is the language of choice, so some experience would be beneficial but is not required.

Anubis will be build on top of a custom written dynamic recompiler, which will also support multiple architectures. The GPU emulation will use wgpu and spir-v to talk to the host gpu.
